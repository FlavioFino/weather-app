# WECS Weather App 🌤

A serverless weather application built at the WECS Weather App Workshop.

**Live demo:** https://weather-app-fino.vercel.app

## Features
- Current weather conditions for any city worldwide
- 5-day forecast
- Air quality index
- Deployed serverlessly on Vercel

## API Endpoints
| Endpoint | Description |
|----------|-------------|
| `/api/weather?city=Victoria` | Current conditions |
| `/api/forecast?city=Victoria` | 5-day forecast |
| `/api/airquality?city=Victoria` | Air quality index |

## Tech Stack
Node.js · OpenWeatherMap API · Vercel · Vanilla HTML/CSS/JS
